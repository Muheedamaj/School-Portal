# School-Portal
A database-driven school management portal built using SQL along with supporting backend and frontend technologies. This project focuses on securely managing student records, teacher details, academic information, and portal authentication through a structured relational database.

=> Features

* Database & SQL Functionalities

  Well-structured relational database with normalized tables for students, teachers, classes, and academic records.

  Primary & Foreign Keys used to maintain secure and accurate relationships between tables.

  Efficient SELECT queries used to fetch student profiles, attendance, class lists, and performance details.

  JOIN queries applied to combine related academic and personal data into meaningful output.

  SQL-based authentication to validate login credentials securely.

  CRUD operations (Insert, Update, Delete) for admin-level management.

* Additional Tools & Technologies

  Backend logic implemented using Python/Django (or your chosen backend).

  Frontend created using HTML, CSS, and JavaScript for clean UI and smooth data display.

  Optional styling tools like Bootstrap used for responsive layouts.

=> Technologies Used

* SQL — Core data structuring, querying & relationship management

* Python/Django — Backend server logic & database connection

* HTML5 — Clean and semantic front-end structure

* CSS3 — Page styling & responsive layout

* JavaScript (ES6) — Interactive UI elements & dynamic data display
